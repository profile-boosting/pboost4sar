
# pboost4sar


## Installation

You can install the development version of pboost4sar like so:

``` r
library(remotes)
install_git("https://github.com/paradoxical-rhapsody/pboost4sar.git")
```


## Examples

This is a basic example which shows you how to solve a common problem:

``` r
library(pboost4sar)
example(psar)
```
